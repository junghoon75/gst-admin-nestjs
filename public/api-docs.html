GST Admin API 문서

버전: 1.0.0

AuthController

기본 경로: /auth
인증 관련 API 컨트롤러
POST /auth/login
사용자 로그인 API
이메일과 비밀번호를 사용하여 로그인하고 JWT 토큰을 발급받습니다.
요청 본문 (LoginDto)
필드 타입 필수 예시
email string Y user@example.com
password string Y password123
응답
상태 코드 설명 타입
200 로그인 성공 { accessToken: string, refreshToken: string }
401 인증 실패 -
POST /auth/refresh
토큰 갱신 API
리프레시 토큰을 사용하여 새로운 액세스 토큰을 발급받습니다.
요청 본문 (RefreshTokenDto)
필드 타입 필수 예시
refreshToken string Y eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
응답
상태 코드 설명 타입
200 토큰 갱신 성공 { accessToken: string, refreshToken: string }
401 유효하지 않은 리프레시 토큰 -
POST /auth/logout
로그아웃 API
사용자 로그아웃을 처리합니다.
인증: JWT
응답
상태 코드 설명 타입
200 로그아웃 성공 -
401 인증되지 않은 사용자 -
GET /auth/profile
사용자 프로필 조회 API
현재 로그인한 사용자의 프로필 정보를 조회합니다.
인증: JWT
응답
상태 코드 설명 타입
200 프로필 조회 성공 User
401 인증되지 않은 사용자 -
